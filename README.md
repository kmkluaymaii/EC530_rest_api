## FastAPI & FDA API Exercises

This repository, implemented by **Rawisara Chairat** and **Pippi Pi**, contains exercises demonstrating API usage, FastAPI development, and integration of external data.

**Exercise 1:** Consume the FDA Adverse Event API using requests in Python.

**Exercise 2:** Build a FastAPI app for account and note management.

**Final:** Combined version where users can fetch FDA data and save it as a note in their account.
